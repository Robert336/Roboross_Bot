## Commands
### Prefix = `~`
 `~stop` makes the bot leave the voice channel
 
 `~hello [name]` plays specific audio clips associated with the name
 
 Names Supported:
 `david`
 
 `~random` randomly selects a message from the text channel and resends it out of context
